## Design Philosophy

AI Studio's UI must do three things:

1. **Feel fast** â€” instant navigation, no loading spinners, keyboard-first
2. **Show depth without complexity** â€” powerful features that don't overwhelm
3. **Be the reason people choose us** â€” OpenClaw is terminal-only. Our UI IS the moat.

### Design References

The UI should feel like a blend of:
- **Linear** â€” clean, fast, keyboard-driven project management
- **Vercel Dashboard** â€” professional dark theme, clear hierarchy
- **Chrome DevTools** â€” dense but navigable technical information
- **Raycast** â€” command palette as primary navigation
